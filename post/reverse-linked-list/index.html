Reverse Linked List

16 Apr 2021
ProblemSolving

Problem_Link

Reverse a singly linked list.

Example:

Input: 1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL

Follow up:

A linked list can be reversed either iteratively or recursively. Could you implement both?

My Answer

My code

class Solution {
     public ListNode reverseList(ListNode head) {


        Stack<ListNode> stack = new Stack<>();

        while (head != null) {
            stack.push(head);
            head = head.next;

        }

        ListNode rtnHead = new ListNode();
        ListNode temp = rtnHead;

        while (!stack.isEmpty()) {
            temp.next = stack.pop();
            temp = temp.next;
        }
        temp.next=null;


        return rtnHead.next;
    }
}

Iterative

public ListNode reverseList(ListNode head) {
    ListNode prev = null;
    ListNode curr = head;
    while (curr != null) {
        ListNode nextTemp = curr.next;
        curr.next = prev;
        prev = curr;
        curr = nextTemp;
    }
    return prev;
}

Complexity analysis

Recursive

public ListNode reverseList(ListNode head) {
    if (head == null || head.next == null) return head;
    ListNode p = reverseList(head.next);
    head.next.next = head;
    head.next = null;
    return p;
}

Recursive process

Complexity analysis

  • Time complexity : $O(n)$
  • Space complexity : $O(n)$
